# Currency Duo

Part of the Fridge Combo project. A two-way, real-time currency converter designed to eliminate cognitive load during international travel and shopping.

## 1 Tool / 1 Action

Instantly convert between two selected currencies. **Currency Duo** automatically fetches the latest exchange rates and provides a seamless, bidirectional input experience.

## Philosophy
* **1 Tool, 1 Action.**
* **No Login / No Ads / No Tracking**
* **Browser-only Privacy:** Rates are fetched directly by your browser. No personal data is ever stored or transmitted.

## How to Use
1. **Setup:** Select your two target currencies (e.g., MYR and JPY). The tool will remember your choice for next time.
2. **Convert:** Type a value into *either* input field. The other field will update instantly.
3. **Read:** Large, bold numbers with thousands-separators (commas) for easy reading, plus a rounded "Approx" value for quick mental checks.
4. **Swap:** Tap the [ ⇄ ] icon to flip the currency positions.
5. **Clear:** Tap the [ Clr ] button to empty the fields and start a new calculation.

## Real-time Sync
* Fetches the latest rates from `open.er-api.com` on startup.
* Works offline using the last cached rates if a connection is unavailable.
